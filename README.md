# Mowatt Marine & Associates

Static redesign for the Mowatt Marine & Associates website.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive visual design
- `script.js` - mobile navigation and header behavior
- `assets/` - logo and selected site imagery
- `review-notes.md` - design review notes from the original GoDaddy site

## Preview

Open `index.html` directly in a browser, or run a local server:

```sh
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173/`.
